# Junk Food Facts & Fitness Screener 🍔💻

My very first vanilla web application built completely from scratch during my Phase 1 Full Stack Development internship with DecodeLabs! 

This project is an interactive, responsive frontend dashboard where users can browse, text-search, and filter real-time nutritional metrics for popular treats, alongside using a custom-built BMI calculator.

## 🛠️ Features
* **Zero Frameworks:** Built entirely without Bootstrap, Tailwind, or React—just pure, native web foundations.
* **Semantic HTML5:** Uses clean layout tags like `<header>`, `<nav>`, `<main>`, and `<article>` to keep the document structure accessible.
* **Responsive Layout:** Powered by CSS Grid and Flexbox to transition smoothly from a mobile view to a 3-column desktop matrix.
* **Vanilla JavaScript Logic:** Drives the live keyword search bar and category filters instantly using custom data attributes without page reloads.

## 🎨 Design Theme
The layout implements a clean, industrial palette requirement:
* **Background:** Moonlit Grey (`#F2F0EA`)
* **Highlights & States:** Ethereal Blue (`#A0D4E0`)
* **Borders & Accents:** Mocha Mousse (`#A8856F`)
