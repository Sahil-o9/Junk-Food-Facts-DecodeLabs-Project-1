# Junk Food Facts & Fitness Screener 🍔💻

My first vanilla web application built completely from scratch during my Phase 1 Full Stack Development internship with DecodeLabs!

This is an interactive, responsive dashboard where users can browse, search, and filter nutritional data for popular snacks, plus calculate their BMI with a custom-built tool. Built with **zero frameworks**—just pure HTML5, CSS3, and vanilla JavaScript.

> **[Live Demo](https://sahil-o9.github.io/Junk-Food-Facts-DecodeLabs-Project-1/)**

---

## ✨ Features

### 🎯 Core Functionality
- **Real-Time Text Search:** Instantly filter foods by name or description with live keyword matching
- **Category Filtering:** Browse by food categories (snacks, beverages, desserts, etc.)
- **BMI Calculator:** Custom-built fitness screener with instant health metrics
- **Nutritional Data Display:** View calories, fats, proteins, and other key metrics

### 💻 Technical Highlights
- **Zero Frameworks:** No Bootstrap, Tailwind, or React—built on native web APIs
- **Semantic HTML5:** Accessible markup with `<header>`, `<nav>`, `<main>`, `<article>`, and ARIA attributes
- **Responsive Design:** Mobile-first CSS with Grid and Flexbox for smooth scaling from mobile to desktop (3-column layout)
- **Vanilla JavaScript:** Custom event handlers and DOM manipulation without external libraries
- **Optimized Performance:** Minimal dependencies = faster load times
